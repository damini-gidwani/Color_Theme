# Color Theme Switcher

A simple JavaScript project that allows users to switch between different color themes by clicking on colored buttons.

## Features

* Change the webpage background color dynamically
* Update heading text color based on the selected theme
* Add a shadow effect to the heading
* Built using HTML, CSS, and JavaScript
* Beginner-friendly DOM manipulation project

##  Preview

Click on any color box to instantly change the page theme.

Available themes:

* Grey
* White
* Yellow
* Green
* Pink
* Purple

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
